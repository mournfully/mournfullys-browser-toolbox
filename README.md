### build from source
```shell
git clone https://mournfully/chrome-extension
cd chrome-extension
npm install
npm run build
# In chrome:
# - go to chrome://extensions
# - enable developer mode
# - load unpacked ~/Downloads/chrome-extension/dist 
```
