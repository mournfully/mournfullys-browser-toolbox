### build from source
```shell
cd ~/Downloads/
git clone https://mournfully/mournfullys-browser-toolbox
cd chrome-extension
npm install

npm run build
# In chrome:
# - go to chrome://extensions
# - enable developer mode
# - load unpacked ~/Downloads/mournfullys-browser-toolbox/dist 
```
